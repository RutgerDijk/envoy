# Finalize Command

Use the envoy:finishing-branch skill to complete this branch and create a PR.

**This will:**
1. Run full 4-layer review
2. Add docstrings to public APIs
3. Sync wiki documentation
4. Run final verification
5. Create pull request

**Preconditions:**
- Must be on a feature branch (not main/master)
- Working directory must be clean
- Tests must be passing

If preconditions are not met, the skill will guide you to resolve them first.
